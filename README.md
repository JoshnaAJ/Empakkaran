### Project Description
*"Vayaru niranjaal mathram poraa… recordum nirakkanam. — Empakkaraa!!!"*

### The Problem (that doesn't exist)
Burps go unnoticed.
No one appreciates the true art, length, and power of a well-executed burp.
Society is missing a centralized burp documentation system to track, celebrate, and analyze burps in real-time.

### The Solution (that nobody asked for)
We present Empakkaran BurpLog – the world’s most groundbreaking app for logging, rating, and sharing your burps.

Automatic burp detection 🎙️

Real-time burp length timer ⏱️

Leaderboard of legendary burpers 🏆

Confetti when you break your personal best 🎉

## Technical Details
### Technologies/Components Used
For Software:
-  JavaScript, HTML, CSS
-  React (frontend), Node.js + Express (backend)
- Tone.js (funny burp sound effects)

Canvas Confetti (celebrations)

Web Speech API (burp detection)
- Git, VS Code, npm

For Hardware:
Microphone (to detect burps)

Laptop/PC/Phone (to run the app)

Your digestive system 🍔➡️💨

### Implementation
For Software:
